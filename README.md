# OmadaSentimentApp

# App Review Scraper & Downloader

A **Streamlit web app** to search mobile apps on **Apple App Store** and **Google Play**, scrape user reviews, and download them as CSV files. Supports scraping up to **500 reviews per app**, with a URL fallback for Google Play apps that may not appear in search results.  

---

## Features

- Search apps by name on **Apple App Store** or **Google Play**.  
- Scrape reviews including:
  - Review date  
  - Rating  
  - Review content  
  - Username  
- Optional **Google Play URL input** for apps not found in search.  
- Download reviews as a **CSV file**.
